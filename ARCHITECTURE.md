## Overview

**Code Share** is a secure, real-time code sharing platform built with Next.js 16. It enables developers to create, share, and collaborate on code snippets with granular access controls.

### Key Features

- ğŸ” **Secure Authentication** - Google OAuth via NextAuth.js
- ğŸ“ **Rich Code Editor** - Monaco Editor with syntax highlighting
- ğŸ”„ **Auto-Save** - Debounced auto-save with beacon API fallback
- ğŸ›ï¸ **Access Control** - Public/Private visibility & collaborative editing
- ğŸ“Š **Storage Quotas** - User file limits with admin overrides
- ğŸ¨ **Modern UI** - Tailwind CSS v4 with custom design system

## Design Patterns Used

| Pattern                 | Implementation               | Purpose                       |
| ----------------------- | ---------------------------- | ----------------------------- |
| **Singleton**           | `lib/db.ts`                  | Single MongoDB connection     |
| **Barrel Exports**      | `index.ts` files             | Clean imports                 |
| **Container/Presenter** | `FileEditor` / UI components | State/presentation separation |
| **Custom Hooks**        | `useAutoSave`, etc.          | Reusable stateful logic       |
| **Server Actions**      | `app/actions/`               | Type-safe RPC mutations       |
| **Enum Pattern**        | `enums/`                     | Type-safe constants           |
| **Strategy Pattern**    | Permission functions         | Flexible authorization        |

## Performance Considerations

1. **Server Components by Default** - Reduces client bundle size
2. **Connection Caching** - MongoDB singleton prevents cold starts
3. **Debounced Auto-Save** - 1500ms debounce reduces API calls
4. **Suspense Loading** - Graceful loading states
5. **Revalidation** - Targeted `revalidatePath` instead of full cache clear
6. **Monaco Lazy Loading** - Editor loads only when needed
